# 群、李群与李代数入门学习计划

> 目标：从“线性代数 + 微积分”出发，建立对群、李群和李代数的最基础直觉，能说清定义、常见例子和它们为什么自然。
> 总时长：7 天，每天 2 小时左右，稳一点就够。

## 设计思路

这套路线先把“群”压成一句人话：一套能组合、能撤回、还能保持结构的操作。然后把这个想法升级到“连续变化的群”，也就是李群，最后再看李代数：它可以先理解成李群在“什么都没做”的位置附近的线性影子。你已经会矩阵、函数、极限，所以这里不走抽象代数的完整证明链，只抓最核心的概念和画面感。

## 怎么学

1. 先打开打卡面板，勾任务，保住连续性。
2. 每天只挑 1 到 2 个任务做深，不要追求一口气读完。
3. 每做完一条，都试着用自己的话讲一遍。
4. 卡住时先回到“思考题”，不要急着跳下一段。

## 可选补充

- 线性代数里最该先复习的是矩阵乘法、可逆矩阵、正交矩阵、行列式。
- 微积分里最该先复习的是函数连续、导数、参数变化的直觉。
- 如果想再往下走，再去看李括号、矩阵指数和指数映射的严格定义。
